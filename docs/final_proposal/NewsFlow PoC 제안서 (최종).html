🗄️ PostgreSQL 3개 테이블

1️⃣ selectors

사이트별 CSS Selector 저장

핵심 컬럼:

  • • site_name: 'yonhap'
  • • title_selector: 'h1.tit01'
  • • body_selector: 'article.article-wrap01'
  • • site_type: 'ssr' or 'spa'

사이트 리뉴얼 → UPDATE만!

2️⃣ crawl_results

크롤링 결과 저장

핵심 컬럼:

  • • url: 중복 방지 (UNIQUE)
  • • title, body, date
  • • quality_score: 0-100
  • • crawl_mode: 'scrapy' | '2-agent'

UC1/UC2/UC3 모두 저장

3️⃣ decision_logs

2-Agent 분석 과정 (JSONB)

핵심 컬럼:

  • • gpt_analysis: JSONB
  • • gemini_validation: JSONB
  • • consensus_reached: bool
  • • retry_count: int

AI 의사결정 투명성

📄 decision_logs JSONB 구조 예시:

{
  "gpt_analysis": {
    "selectors": {"title": "h1.new-class", "body": "article.new-body"},
    "confidence": 0.85,
    "reasoning": "사이트가 2025-10-28 리뉴얼했습니다..."
  },
  "gemini_validation": {
    "valid": true,
    "samples": [{"title": "샘플 제목 1", "body_length": 1500}],
    "confidence": 0.9
  }
}

→ PostgreSQL GIN 인덱스로 JSON 쿼리 최적화
